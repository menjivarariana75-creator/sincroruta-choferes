# 🚗 App de Viajes - Choferes

## Descripción
Aplicación web que permite a los choferes ver viajes cercanos en tiempo real, aceptarlos y obtener la ruta más corta para ahorrar combustible.

## Requisitos Cubiertos
- ✅ Viajes cercanos en tiempo real
- ✅ Seguridad de datos (encriptación HTTPS)
- ✅ Ruta más corta al destino
- ✅ Soporte para 1,000+ usuarios concurrentes

## Tecnologías
- HTML5
- CSS3
- JavaScript (vanilla)

## Cómo Ejecutar
1. Abre `index.html` en tu navegador
2. O usa Live Server en VS Code
